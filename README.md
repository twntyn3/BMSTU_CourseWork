# BMSTU CourseWork

В этом репозитории собрана большая часть моих лабораторных работ, учебных проектов, практических заданий и других материалов, выполненных в процессе обучения в **Московском государственном техническом университете имени Н. Э. Баумана**.

Основное направление обучения — **01.03.02 «Прикладная математика и информатика»** на кафедре **ИУ9 «Теоретическая информатика и компьютерные технологии»** факультета **«Информатика и системы управления»**.

## О репозитории

Репозиторий используется как единое хранилище учебных работ по разным дисциплинам. Здесь могут находиться:

- лабораторные работы;
- курсовые и учебные проекты;
- исходный код программ;
- отчёты и README-файлы;
- материалы для защиты и пояснения к выполненным заданиям.

## Структура

Материалы разложены по папкам в зависимости от дисциплины или темы. Внутри отдельных разделов могут находиться дополнительные README-файлы с описанием конкретных лабораторных работ, среды выполнения и особенностей реализации.

## Примечание

Некоторые тяжёлые бинарные файлы, собранные образы, файлы виртуальных машин, временные файлы и результаты компиляции не добавляются в репозиторий. В GitHub хранятся только те материалы, которые имеют смысл для просмотра, проверки и дальнейшего использования.
