# Лабораторная работа №3 — ReactOS

## Описание

В этой части лабораторной работы был реализован загружаемый драйвер для ReactOS, который выводит в отладочный лог список процессов в системе.

Драйвер получает информацию о процессах с помощью API ядра и выводит основные данные о каждом процессе: идентификатор процесса, идентификатор родительского процесса и имя образа процесса.

## Что было сделано

- создан новый драйвер на основе предыдущей лабораторной работы;
- драйвер подключён к дереву сборки ReactOS;
- реализовано получение списка процессов через ZwQuerySystemInformation;
- реализовано выделение и освобождение памяти под буфер с информацией о процессах;
- учтена ситуация, когда размер буфера может измениться между вызовами;
- результат выводится в debug log;
- драйвер загружается по требованию через Service Control Manager.

## Файлы

- lab3_react.c — исходный код драйвера для ReactOS.

## Проверка результата

После сборки и установки системы драйвер загружался командами:

    sc create lab3driver binPath= C:\ReactOS\system32\drivers\lab3.sys type= kernel
    sc start lab3driver

После запуска в отладочном логе появлялся список процессов системы.

## Среда выполнения

Работа выполнялась с использованием виртуальной машины. Основная работа с VM проводилась на MacBook M2, сборка и подготовка ядра выполнялись самостоятельно.
