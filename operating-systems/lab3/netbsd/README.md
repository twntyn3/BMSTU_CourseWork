# Лабораторная работа №3 — NetBSD

## Описание

В этой части лабораторной работы был реализован загружаемый модуль ядра NetBSD, который выводит список процессов в системный лог.

Для получения информации о процессах используется список процессов ядра NetBSD и макрос для прохода по этому списку.

## Что было сделано

- создан загружаемый модуль ядра NetBSD;
- модуль добавлен в дерево сборки модулей;
- реализован проход по списку процессов;
- для каждого процесса выводится PID, PPID и имя процесса;
- модуль собирается средствами NetBSD;
- модуль загружается командой modload;
- результат проверяется через системный лог.

## Файлы

- lab3_netbsd.c — исходный код модуля ядра для NetBSD.

## Проверка результата

Сборка выполнялась в директории модуля командой:

    make

Загрузка модуля выполнялась командой:

    modload ./lab3.kmod

После загрузки в системном логе появлялся список процессов.

## Среда выполнения

Работа выполнялась внутри виртуальной машины NetBSD. Сборка модуля проводилась средствами самой NetBSD.
