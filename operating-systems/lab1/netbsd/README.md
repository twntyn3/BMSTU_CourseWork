# Лабораторная работа №1 — NetBSD

## Описание

В этой части лабораторной работы была установлена NetBSD в виртуальную машину и настроена среда сборки ядра внутри самой NetBSD.

После настройки исходных кодов ядро было пересобрано с изменением, которое выводит фамилию студента в системный лог во время загрузки.

## Что было сделано

- установлен дистрибутив NetBSD в виртуальную машину;
- настроена среда сборки внутри NetBSD;
- получены исходные коды системы;
- внесено изменение в код ядра;
- пересобрано ядро NetBSD;
- выполнена загрузка с модифицированным ядром;
- результат проверен через системный лог.

## Почему нет файлов ядра

Собранное ядро, образы и файлы виртуальной машины не добавлены в репозиторий, так как это тяжёлые бинарные артефакты, зависящие от конкретной конфигурации виртуальной машины.
