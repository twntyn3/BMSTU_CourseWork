# Лабораторная работа №1 — ReactOS

## Описание

В этой части лабораторной работы была настроена среда сборки ReactOS, собран установочный ISO-образ и выполнена установка системы в виртуальную машину.

После установки было модифицировано ядро ReactOS: в процессе инициализации ядра в отладочный лог выводилась фамилия студента.

## Что было сделано

- установлена и настроена ReactOS Build Environment;
- получены исходные коды ReactOS;
- собран установочный ISO-образ ReactOS;
- ReactOS установлена в виртуальную машину;
- настроен вывод отладочного лога;
- изменено ядро ReactOS;
- модифицированное ядро заменено в установленной системе;
- результат проверен по debug log.

## Почему нет исходных файлов

В этой части основной результат — настроенная среда, собранный образ и модифицированное ядро.

Само собранное ядро и ISO-образ не добавлены в GitHub, потому что это тяжёлые бинарные файлы, которые зависят от конкретной среды сборки.
