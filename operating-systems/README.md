# Лабораторные работы по операционным системам

В этой папке находятся материалы по лабораторным работам курса «Операционные системы».

Работы выполнялись для двух операционных систем:

- ReactOS — открытая Windows NT-подобная операционная система;
- NetBSD — UNIX-подобная операционная система с открытым исходным кодом.


## Среда выполнения

Лабораторные работы выполнялись с использованием виртуальных машин.

Основная работа с виртуальными машинами проводилась на MacBook M2. Сборка ядер и подготовка систем также выполнялись самостоятельно, в том числе на компьютере с Windows 11.

## Почему в репозитории нет образов и собранных ядер

ISO-образы, файлы виртуальных машин, собранные ядра и бинарные драйверы не добавлены в репозиторий, так как они имеют большой размер, зависят от конкретной среды сборки и не нужны для просмотра исходного кода на GitHub.

В репозитории оставлены README-файлы с описанием выполненных частей лабораторных работ и исходные файлы там, где они действительно имеют смысл.
