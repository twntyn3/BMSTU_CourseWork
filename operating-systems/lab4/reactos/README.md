# Лабораторная работа №4 — ReactOS

## Описание

В этой части лабораторной работы был реализован загружаемый драйвер ReactOS для работы с виртуальной памятью в режиме ядра.

Драйвер резервирует область виртуальной памяти, фиксирует часть страниц в физической памяти, выводит информацию о физических адресах и значениях PTE, после чего освобождает выделенную память.

## Что было сделано

- создан новый драйвер на основе предыдущих лабораторных работ;
- реализовано резервирование 10 страниц виртуальной памяти;
- реализовано выделение физических страниц для первых 5 страниц;
- получены и выведены физические адреса страниц;
- получены и выведены значения PTE;
- после завершения работы память освобождается;
- результат выводится в debug log.

## Файлы

- lab4_react.c — исходный код драйвера для ReactOS.

## Проверка результата

После сборки и установки системы драйвер загружался командами:

    sc create lab4driver binPath= C:\ReactOS\system32\drivers\lab4.sys type= kernel
    sc start lab4driver

После запуска в отладочном логе появлялась информация о выделенной виртуальной памяти, физических адресах страниц и значениях PTE.

## Среда выполнения

Работа выполнялась с использованием виртуальной машины. Основная работа с VM проводилась на MacBook M2, сборка и подготовка ядра выполнялись самостоятельно.
