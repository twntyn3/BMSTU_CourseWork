# Лабораторная работа №4 — NetBSD

## Описание

В этой части лабораторной работы был реализован загружаемый модуль ядра NetBSD для работы с виртуальной памятью.

Модуль выделяет память в адресном пространстве ядра, получает информацию об адресах страниц и выводит результат в системный лог.

## Что было сделано

- создан загружаемый модуль ядра NetBSD;
- реализовано выделение памяти в режиме ядра;
- реализован проход по выделенным страницам;
- получены и выведены адреса страниц;
- после завершения работы память освобождается;
- результат проверен через системный лог.

## Файлы

- lab4_netbsd.c — исходный код модуля ядра для NetBSD.

## Проверка результата

Сборка выполнялась командой:

    make

Загрузка модуля выполнялась командой:

    modload ./lab4.kmod

После загрузки в системном логе появлялась информация о работе с виртуальной памятью.

## Среда выполнения

Работа выполнялась внутри виртуальной машины NetBSD. Сборка модуля проводилась средствами самой NetBSD.
