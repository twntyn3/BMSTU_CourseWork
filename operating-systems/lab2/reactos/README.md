# Лабораторная работа №2 — ReactOS

## Описание

В этой части лабораторной работы был разработан простой загружаемый драйвер для ReactOS, совместимый с архитектурой драйверов Windows NT.

Драйвер загружался по требованию и при запуске выводил фамилию студента в отладочный лог.

## Что было сделано

- создан новый модуль драйвера в дереве исходных кодов ReactOS;
- добавлены файлы сборки для подключения драйвера к системе;
- реализована минимальная структура драйвера;
- добавлен вывод сообщения в debug log;
- собран новый ISO-образ ReactOS;
- драйвер загружен в установленной системе;
- результат проверен по отладочному логу.

## Почему нет бинарных файлов

Собранный драйвер, ISO-образ и файлы виртуальной машины не добавлены в репозиторий, так как являются результатами сборки.
