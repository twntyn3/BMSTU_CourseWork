# Лабораторная работа №2 — NetBSD

## Описание

В этой части лабораторной работы был реализован простой загружаемый модуль ядра NetBSD.

Модуль собирался внутри дерева исходных кодов NetBSD и загружался командой modload. При загрузке модуль выводил фамилию студента в системный лог.

## Что было сделано

- создан исходный файл загружаемого модуля ядра;
- создан Makefile для сборки модуля;
- реализована минимальная функция обработки команд модуля;
- выполнена сборка kmod-файла;
- модуль загружен командой modload;
- результат проверен через системный лог.

## Почему нет kmod-файла

Собранный kmod-файл не добавлен в репозиторий, так как является бинарным результатом сборки и зависит от версии NetBSD.
